# 价格影响等级系统

灵感来自炉石传说的稀有度系统,根据价格波动幅度划分影响等级。

## 等级定义

| 等级 | 图标 | 颜色 | 波动范围 | 说明 |
|---|---|---|---|---|
| **传说级** | 💎 | 橙色 `#ff8800` | ≥ 20% | 市场震荡,重大变革。如CS2正式发布导致刀具价格暴涨35% |
| **史诗级** | ⚜️ | 紫色 `#a335ee` | 10-20% | 显著影响,值得关注。如军械库上线导致步枪涨15% |
| **稀有级** | 💠 | 蓝色 `#0070dd` | 5-10% | 中等影响,稳健投资。如手套类受十合一更新影响涨7% |
| **普通级** | ⚪ | 灰色 `#9d9d9d` | < 5% | 轻微波动,影响有限。如刚需武器AK-47仅涨3% |

## 视觉效果

### 徽章样式
- 渐变色背景
- 发光阴影效果
- 呼吸动画(brightness 1.0 ↔ 1.2)
- 传说级最耀眼,普通级最低调

### 应用场景

**1. 分类总体影响**
```
🔪 刀具
  ↓ -12.5% 💎 传说级影响
```

**2. 子分类详细影响**
```
爪子刀 💎     ↓ -12.5%
蝴蝶刀 💎     ↓ -15.3%
M9刺刀 ⚜️     ↓ -10.8%
```

## 计算逻辑

```javascript
function getImpactLevel(percentage) {
    const absPercent = Math.abs(percentage);

    if (absPercent >= 20) return { level: 'legendary', icon: '💎', color: '#ff8800' };
    if (absPercent >= 10) return { level: 'epic', icon: '⚜️', color: '#a335ee' };
    if (absPercent >= 5)  return { level: 'rare', icon: '💠', color: '#0070dd' };
    return { level: 'common', icon: '⚪', color: '#9d9d9d' };
}
```

## 设计理念

1. **直观性**: 颜色和图标一眼识别影响程度
2. **游戏化**: 借鉴炉石传说,降低理解成本
3. **可扩展**: 未来可以添加"神话级"等更高等级
4. **美观性**: 渐变色+发光效果,提升视觉体验

## 与原系统对比

| 旧系统(可信度) | 新系统(影响等级) | 优势 |
|---|---|---|
| 高/中/低可信度 | 传说/史诗/稀有/普通 | 更直观,与价格波动直接关联 |
| 基于数据来源质量 | 基于实际影响幅度 | 更符合用户关心的"影响有多大" |
| 绿/黄/红单调色 | 橙/紫/蓝/灰游戏化 | 更有视觉冲击力 |
