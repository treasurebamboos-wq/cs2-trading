## 🔌 数据接口设计

### API Endpoints (预留,方便对接你的价格项目)

```javascript
// 当前: 使用模拟数据
GET /api/items?category=rifle&rarity=classified

// 未来: 对接真实数据项目
GET /api/v2/items?category=rifle&rarity=classified
// 只需切换endpoint,数据格式保持一致

// 价格历史
GET /api/prices/history/{itemId}?range=30d
返回: { dates: [], prices: { buff: [], youpin: [], steam: [] }}

// 实时价格
GET /api/prices/realtime?items=id1,id2,id3
返回: { id1: {buff: 125.5, change24h: 3.2}, ... }

// 搜索
GET /api/search?q=AK-47&limit=10
返回: [{ id, name, price, image }, ...]

// 排行榜
GET /api/rankings?type=gainers&period=24h&limit=50
返回: [{ rank, item, change }, ...]
```
